Getting Started

What is Aura Create?

Aura Create serves as an administrative web application for Aura's technology. It offers the ability to manage and observe data from multiple. With this website, an administrator should be able to: add, edit, and remove any data that is associated with an organization. Aura Create is a way to monitor Aura Explore and to help provide the best user experience possible.

What is a Beacon?

A beacon is Aura's patented technology that enables precise geospatial mapping. They are also able to transmit bluetooth to mobile devices what allows for fast content delivery based on a users location relative to objects.

What is an Object?

Objects are a way of binding tangible objects into the digital world thanks to the usage of beacons. They can resemble any real object which allows for great flexibility and customization. Objects are able to store various media types such as: images, audio, video, 3D, and 360° images. These media files are stored as data called "Assets" that are binded to an object.

What is an Asset?

As previously mentioned previously, assets are media files including but not limited to: images, audio, video, 3D, and 360° images. They enable us to virtually store an objects true nature and allows Aura Explore users to experience them in many exciting ways.
