Administrator Guide

Summary

Aura Create serves as an administrative web application for Aura's technology. It offers the ability to manage and observe data from multiple organization's.

Navigation: An important thing to remember when navigating throughout the web-app is to consider which organization is currently selected. The selected organization is highlighted in the list of all organizations; text indicating the organization is also shown under the user icon in the top right corner of the web page.

With a selected organization, their respective dashboard, beacons, objects, and settings can be brought into focus through the navigation bar at the top of the application.
